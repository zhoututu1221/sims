## 基于JavaEE技术的工资信息管理系统设计与实现，使用了SSM框架

> 技术栈

Java、Spring、SpringMVC、MyBatis、MySQL、Maven、LayUI、JQuery、Echarts...



> 主要功能

这是一个员工工资管理系统

系统分为员工端和管理员端两个部分

员工可以登录后查看自己的工资情况

管理员可以对员工工资的发放查询等进行管理操作
